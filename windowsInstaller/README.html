Welcome!

Thank you for installing the application. To finish setting everything up, please follow the simple steps for your operating system.

Windows

Step 1: Open Docker

  1. Find the Docker Desktop icon on your Desktop.
  2. Right-click the icon and choose Run as administrator.
  3. If you see a pop-up that asks, “Do you want to allow this app to make changes to your device?”, click Yes.
  4. Wait for Docker to fully start before moving on.

Step 2: Run the Program

  1. Find the file called fEMR.exe on your Desktop or in the Start Menu.
  2. Double-click on the file.
  3. If asked to confirm anything, click Yes.

Having Trouble?

If you see a message that says scripts are blocked:

  1. Right-click the Start Menu and choose Windows PowerShell (Admin).
  2. Copy and paste this command, then press Enter:
    3. Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
  3. When asked, type Y and press Enter.
  4. Close the window, then try running fEMR.exe again.

Mac

Step 1: Open femr

  1. Open the Applications folder.
  2. Find the app called femr.
  3. Double-click to run it.
  4. If you can't find it, press Command + Space to open Spotlight Search.
  5. Type femr and press Enter.

If your Mac asks for permission to run the app, click Open or Allow.
